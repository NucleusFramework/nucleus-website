## Apps multi-fenĂŞtres

```kotlin

    DecoratedWindow(onCloseRequest = ::exitApplication, title = "Main") {
        TitleBar { _ -> /* â€¦ */ }
        Button(onClick = { showInspector = true }) { Text("Open inspector") }

            onCloseRequest = { showInspector = false },
            title = "Inspector",

            TitleBar { _ -> Text("Inspector") }
            InspectorContent()
        }
    }

```
